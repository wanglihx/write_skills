##写作规范
这是学术论文中的常见约定：
  - 花体字母（$\mathcal{M}$, $\mathcal{V}$, $\mathcal{S}$）→ 集合、序列
  - 普通斜体（$M_t$, $I_t$, $\theta_t$）→ 集合中的元素

  所以 $\mathcal{M} = {M_t}_{t=1}^{T}$ 的意思是：
  "掩码序列 $\mathcal{M}$ 由从 $t=1$ 到 $T$ 的所有单帧掩码 $M_t$ 组成"

  这是故意设计成不同样式的，便于读者区分整体和元素。
